# rbx-uk-friends
working proof of concept of viewing Roblox UK residents friends lists

# how it works
* this uses the GetFriendAsync function in roblox studio, they forgot to make it not display UK friends lists
* the code in this repo provides a working proof of concept including making it communicate externally
